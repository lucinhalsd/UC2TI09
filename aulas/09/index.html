Atag img em html é usada para incorporar imagem em uma pagina web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento e contem apenas atributos que definem o conteudo

Atributo principal:

  • src (source): define o caminho da umagem que for exibida. pode ser uma URL externa ou um caminho relativo dentro do servidor.
